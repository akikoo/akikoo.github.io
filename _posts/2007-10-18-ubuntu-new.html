Could 2008 be the year of the Linux desktop? With the latest release of Ubuntu Linux (7.10), I believe for me it will be the year when I finally drop Windows and continue using only Mac OSX and Linux. At home, that is. At work, it might be a different story.

As a former Gentoo Linux user and current Ubuntu enthusiast, I have to say that I'm quite thrilled with latest developments in desktop Linux. One thing is for sure: I won't touch Windows Vista if I have a choice.

See a Wired review on Ubuntu, and go on, download and install Ubuntu -- it's very easy, and free. You probably won't look back.
